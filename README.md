# Social-Media-Engagement-Analysis

## 📌 Objective
Analyze social media engagement metrics like likes, comments, and shares to understand user interaction patterns.

## 📂 Dataset
- Source: CSV file
- Features:
  - Likes
  - Comments
  - Shares
  - Post Type
  - Date

## 🛠 Tools Used
- Python
- Pandas
- Matplotlib / Seaborn
- Jupyter Notebook

## 📊 Analysis Performed
- Engagement comparison
- Trend analysis
- Best posting time identification

## 📈 Results
Insights into which content type performs best and optimal posting times.
